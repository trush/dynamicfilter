Detailed Description

Stores variables needed in the chao_estimator() function.

def chao_estimator (   self)

estimator returns true if the current size of the secondary list is within a certain threshold of the total size of the list (according to the chao estimator) and false otherwise.

Remarks
Uses the Chao92 equation to estimate population size during enumeration. To understand the math computed in this function see: http://www.cs.albany.edu/~jhh/courses/readings/trushkowsky.icde13.enumeration.pdf
