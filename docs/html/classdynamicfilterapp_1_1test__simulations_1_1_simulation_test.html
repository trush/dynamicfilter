def get_sample_answer_dict (   self,
  filename 
)

Reads in a file of pre-gathered Mechanical Turk HITs and makes a dictionary where the key is a IP_Pair and the value is a list of all the HITs answers for that IP_Pair.

This list is the set that our simulations can sample answers from. At present, the csv file downloaded from Mechanical Turk must be copied and then edited to only include the four columns of data that we use here.

Parameters
filenameThe appropriate real data file generated from AMT. Must be the correct file for the ITEM_TYPE to work properly.
Returns
The dictionary with keys that are IP pairs and values that are arrays containing the possible worker responses, True or False, for the IP pair (from real AMT data).
